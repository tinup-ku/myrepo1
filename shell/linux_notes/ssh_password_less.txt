

ssh passwordless:

Client:
   on the client ( which will be doing ssh to a server )
   if you can't find .ssh/id_rsa.pub
   generate keys
     ssh-keygen -t rsa
   copy the contents of file:
     .ssh/id_rsa.pub

Server:
  ( you will login to this host )
   copy the content from above client to file:
      .ssh/authorized_keys

  
  Make sure dir .ssh and file - authorized_keys have permissions set to 700
